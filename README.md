# New Tab Extension

Simple new-tab replacement for Chromium browsers.

## Download

Option 1: Clone with git

```bash
git clone <REPO_URL>
```

Option 2: Download ZIP

1. Open the repository on GitHub.
2. Click "Code" > "Download ZIP".
3. Unzip it to a folder on your computer.

## Install in Chrome (Unpacked)

1. Open `chrome://extensions`.
2. Enable "Developer mode" (top right).
3. Click "Load unpacked".
4. Select the repo folder (the one with `manifest.json`).
5. Open a new tab to see the extension.

## Reload after changes

If you edit any files, return to `chrome://extensions` and click "Reload" on the extension, then open a new tab again.
